### Final Project

**RPG Skill Tree Planner**

A skill tree from an RPG game where you can plan character builds. For example to get the fireball skill you need 5 points in magic and the spark skill. This project uses a tree / graph to model skill dependancies, a dictionary to lookup a kill by name in O(1) time, and a stack for an undo feature when allocating points.
